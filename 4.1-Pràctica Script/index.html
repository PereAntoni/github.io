PRÀCTICA SCRIPT

La pràctica consisteix en el joc endevina nombre. La màquina "pensa" un nombre i l'usuari l'ha d'endevinar.

Descripció

  1. Pantalla on es pot iniciar/sortir del joc.
  2. Pantalla on es demana el nom del jugador.

  3. Comença partida

    1. El programa genera un nombre 'pensat' entre 0 i 100.
    2. L'usuari escriu un nombre 'x' entre 1 i 99.
    3. El programa contesta si és més alt o més baix del nombre 'pensat'.
    4. El programa compara si 'x' és igual a 'pensat'.
      1. Si x=pensat segueix al següent punt
      2. Sino: pasa al punt 2

  4. El programa genera un nobre ramdom
