Qüestionari sobre configuració del servidor Apache: Errors comuns

  1. Quin error es pot cometre en configurar un servidor Apache per HTTPS sense tenir un certificat vàlid?
  2. A) El servidor no funcionarà correctament.
  3. B) Els fitxers no es carregaran en HTTPS.
  4. C) HTTPS funciona, però amb un avís de seguretat.

  5. D) HTTPS es configura automàticament sense problemes.

  6. Quina d'aquestes opcions pot provocar que Apache no arrenqui després d’una modificació al fitxer httpd.conf?

  7. A) Ús incorrecte d’una directiva sense incloure el mòdul requerit.
  8. B) Arxius de log no configurats.
  9. C) Absència del fitxer de configuració SSL.

  10. D) Instal·lar Apache sense permisos d'usuari root.

  11. Quin error apareix sovint si el port 80 o 443 està ocupat per un altre servei?

  12. A) Error 404
  13. B) Error de sintaxi
  14. C) Error de vinculació de socket (Address already in use)

  15. D) Error d’autenticació

  16. Quina comanda hauries d’executar després de modificar apache2.conf per aplicar els canvis?

  17. A) service apache2 start
  18. B) service apache2 stop
  19. C) systemctl reload apache2

  20. D) apache2 reset

  21. Què indica l’error “403 Forbidden” quan es carrega una pàgina Apache?

  22. A) El fitxer de configuració httpd.conf té un error.
  23. B) El directori arrel té permisos restrictius.
  24. C) El port 80 no està obert.

  25. D) La IP no està configurada.

  26. Quin error comet un usuari si es troba amb “500 Internal Server Error” després de canviar htaccess?

  27. A) Ha establert permisos massa alts al directori.
  28. B) No ha activat el mòdul mod_rewrite.
  29. C) Ha editat httpd.conf en comptes de htaccess.

  30. D) Ha activat el mòdul SSL sense certificat.

  31. Quina és la causa comuna de l’error "AH00558: apache2: Could not reliably determine the server's fully qualified domain name"?

  32. A) Falta la configuració de ServerName a apache2.conf.
  33. B) No està activat el mòdul SSL.
  34. C) El fitxer htaccess té sintaxi incorrecta.

  35. D) Els permisos del servidor són massa restrictius.

  36. Quina comanda serveix per comprovar si el fitxer de configuració Apache té errors de sintaxi?

  37. A) apachectl test
  38. B) apache2 check
  39. C) apachectl configtest

  40. D) systemctl check apache2

  41. Quina d'aquestes configuracions podria generar un error si no s'activa abans en Apache?

  42. A) ServerName
  43. B) DocumentRoot
  44. C) AllowOverride All

  45. D) Listen 80

  46. Quin error es genera si https es configura en un VirtualHost sense tenir el mòdul SSL activat?

    • A) Error 500
    • B) El servidor redirigeix automàticament a HTTP
    • C) Error d'inici del servei per absència del mòdul SSL
    • D) El servidor ignora la configuració i es manté en HTTP
